# Raspberry Pi Security Watchdog

Lightweight Raspberry Pi login and intrusion event monitor with Telegram alerts, designed for self-hosted home servers exposed to the internet.

## Initial scope

- SSH successful login alerts
- SSH failed / pre-auth connection alerts
- Nginx suspicious request monitoring
- Cockpit exposure monitoring
- Flask app exposure checks
- Telegram notifications
- Allowed IP range filtering for LAN, NordVPN Meshnet and Tailscale

## Status

Early local development.
